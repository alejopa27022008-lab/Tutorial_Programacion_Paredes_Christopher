Listas (ArrayList) en Java

Aprende a trabajar con colecciones dinámicas utilizando ArrayList.

Introducción

Los arreglos tienen un tamaño fijo. En muchos casos necesitamos agregar o eliminar elementos mientras el programa está en ejecución.

Para resolver este problema Java ofrece la clase ArrayList.

¿Qué es un ArrayList?

Un ArrayList es una estructura dinámica que permite almacenar múltiples elementos y modificar su tamaño automáticamente. 

import java.util.ArrayList;

Ventajas frente a los Arreglos

Arreglo ArrayList
Tamaño fijo Tamaño dinámico
Menos flexible Más flexible
Difícil insertar elementos Inserción sencilla
Difícil eliminar elementos Eliminación sencilla

Crear una Lista

import java.util.ArrayList;

ArrayList nombres =
new ArrayList<>();

Agregar Elementos

nombres.add("Ana");

nombres.add("Carlos");

nombres.add("Luis");

Mostrar Elementos

System.out.println(nombres);

Salida:

[Ana, Carlos, Luis]

Obtener un Elemento

System.out.println(
    nombres.get(1)
);

Salida:

Carlos

Modificar un Elemento

nombres.set(
    1,
    "Pedro"
);

Eliminar un Elemento

nombres.remove(0);

Tamaño de la Lista

System.out.println(
    nombres.size()
);

Recorrer una Lista

for(String nombre : nombres){

    System.out.println(
        nombre
    );

}

Búsqueda de Elementos

if(
    nombres.contains("Luis")
){

    System.out.println(
        "Existe"
    );

}

Ejemplo Básico Comentado

import java.util.ArrayList;

public class ListaBasica {

    public static void main(String[] args){

        ArrayList ciudades =
        new ArrayList<>();

        ciudades.add("Ambato");
        ciudades.add("Quito");
        ciudades.add("Cuenca");

        for(String ciudad :
            ciudades){

            System.out.println(
                ciudad
            );

        }

    }

}

Resumen

  • ArrayList es una colección dinámica.
  • Permite agregar y eliminar elementos fácilmente.
  • Su tamaño cambia automáticamente.
  • Ofrece métodos como add, remove, get y contains.
  • Es una de las estructuras más utilizadas en Java.
