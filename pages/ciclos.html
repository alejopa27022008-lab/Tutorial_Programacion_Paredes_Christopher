Ciclos (Bucles) en Java

Los ciclos permiten repetir instrucciones varias veces sin necesidad de escribir el mismo código repetidamente.

Introducción

Imagina que necesitas mostrar los números del 1 al 100. Sería muy tedioso escribir 100 instrucciones. Los ciclos permiten automatizar tareas repetitivas.

Conceptos Teóricos

¿Qué es un ciclo?

Un ciclo es una estructura de control que repite instrucciones mientras se cumpla una condición.

Tipos de ciclos en Java

  • for
  • while
  • do-while

Ciclo For

Se utiliza cuando sabemos cuántas veces se repetirá una acción. 

for(inicialización; condición; incremento){
    instrucciones;
}

Ejemplo

for(int i = 1; i <= 5; i++){

    System.out.println(i);

}

Salida:

1
2
3
4
5

Explicación

  • i inicia en 1
  • Se ejecuta mientras i sea menor o igual a 5
  • i aumenta de uno en uno

Diagrama de Flujo del For

flowchart TD A[Inicio] --> B[i = 1] B --> C{ i <= 5 ? } C -->|Sí| D[Mostrar i] D --> E[i++] E --> C C -->|No| F[Fin]

Ciclo While

Se utiliza cuando no conocemos exactamente cuántas repeticiones habrá. 

while(condición){

    instrucciones;

}

Ejemplo

int i = 1;

while(i <= 5){

    System.out.println(i);

    i++;

}

Diagrama While

flowchart TD A[Inicio] --> B[i=1] B --> C{ i <=5 ? } C -->|Sí| D[Mostrar i] D --> E[i++] E --> C C -->|No| F[Fin]

Ciclo Do While

La diferencia es que siempre se ejecuta al menos una vez. 

do{

    instrucciones;

}while(condición);

Ejemplo

int i = 1;

do{

    System.out.println(i);

    i++;

}while(i <= 5);

Diagrama Do While

flowchart TD A[Inicio] --> B[Mostrar i] B --> C[i++] C --> D{ i <=5 ? } D -->|Sí| B D -->|No| E[Fin]

Ejemplo Básico Comentado

public class EjemploFor {

    public static void main(String[] args) {

        // Repetir 10 veces

        for(int i = 1; i <= 10; i++) {

            System.out.println(i);

        }

    }

}

Ejemplo Práctico

Tabla de multiplicar del 7

public class Tabla {

    public static void main(String[] args) {

        for(int i = 1; i <= 10; i++){

            System.out.println(
                "7 x " + i +
                " = " + (7*i)
            );

        }

    }

}

Ejercicios Propuestos

  1. Mostrar los números del 1 al 20 usando for.
  2. Mostrar los números pares del 2 al 100.
  3. Calcular la suma de los números del 1 al 50.
  4. Mostrar una tabla de multiplicar ingresada por el usuario.
  5. Contar cuántos números son múltiplos de 3 entre 1 y 100.

Soluciones

Ejercicio 1

for(int i=1; i<=20; i++){

    System.out.println(i);

}

Ejercicio 2

for(int i=2; i<=100; i+=2){

    System.out.println(i);

}

Ejercicio 3

int suma = 0;

for(int i=1; i<=50; i++){

    suma += i;

}

System.out.println(suma);

Ejercicio 4

Scanner sc = new Scanner(System.in);

int tabla = sc.nextInt();

for(int i=1; i<=10; i++){

    System.out.println(
        tabla + " x " + i +
        " = " + (tabla*i)
    );

}

Ejercicio 5

int contador = 0;

for(int i=1; i<=100; i++){

    if(i % 3 == 0){

        contador++;

    }

}

System.out.println(contador);

Resumen

  • for se usa cuando conocemos las repeticiones.
  • while depende de una condición.
  • do-while se ejecuta al menos una vez.
  • Los ciclos evitan repetir código.
  • Son fundamentales en programación.
