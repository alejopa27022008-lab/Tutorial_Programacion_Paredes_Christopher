Manejo de Archivos en Java

Aprende a leer y escribir archivos TXT y CSV, una habilidad esencial para almacenar información.

Introducción

Los archivos permiten guardar información de manera permanente, incluso después de cerrar un programa.

Son ampliamente utilizados para almacenar usuarios, configuraciones, reportes, estadísticas y bases de datos simples.

Tipos de Archivos Más Utilizados

  • TXT (Texto plano)
  • CSV (Valores separados por comas)
  • JSON
  • XML
  • PDF

Escribir un Archivo TXT

import java.io.FileWriter;

public class EscribirTXT {

    public static void main(String[] args) {

        try {

            FileWriter archivo =
            new FileWriter("ejemplo.txt");

            archivo.write(
                "Hola Mundo"
            );

            archivo.close();

            System.out.println(
                "Archivo creado"
            );

        } catch(Exception e){

            System.out.println(
                e.getMessage()
            );

        }

    }

}

Leer un Archivo TXT

import java.io.BufferedReader;
import java.io.FileReader;

public class LeerTXT {

    public static void main(String[] args) {

        try {

            BufferedReader lector =
            new BufferedReader(
                new FileReader(
                    "ejemplo.txt"
                )
            );

            String linea;

            while(
                (linea =
                lector.readLine())
                != null
            ){

                System.out.println(
                    linea
                );

            }

            lector.close();

        } catch(Exception e){

            System.out.println(
                e.getMessage()
            );

        }

    }

}

¿Qué es un Archivo CSV?

CSV significa Comma Separated Values (Valores Separados por Comas). 

id,nombre,edad

1,Ana,20

2,Luis,22

3,Pedro,18

Escribir un Archivo CSV

import java.io.FileWriter;

public class ExportarCSV {

    public static void main(String[] args) {

        try {

            FileWriter archivo =
            new FileWriter(
                "usuarios.csv"
            );

            archivo.write(
                "id,nombre,edad\n"
            );

            archivo.write(
                "1,Ana,20\n"
            );

            archivo.write(
                "2,Luis,22\n"
            );

            archivo.close();

        } catch(Exception e){

            System.out.println(
                e.getMessage()
            );

        }

    }

}

Leer un Archivo CSV

import java.io.BufferedReader;
import java.io.FileReader;

public class LeerCSV {

    public static void main(String[] args) {

        try {

            BufferedReader lector =
            new BufferedReader(
                new FileReader(
                    "usuarios.csv"
                )
            );

            String linea;

            while(
                (linea =
                lector.readLine())
                != null
            ){

                String[] datos =
                linea.split(",");

                for(String dato :
                    datos){

                    System.out.print(
                        dato + " "
                    );

                }

                System.out.println();

            }

            lector.close();

        } catch(Exception e){

            System.out.println(
                e.getMessage()
            );

        }

    }

}

Buenas Prácticas

  • Siempre cerrar los archivos.
  • Usar try-catch para controlar errores.
  • Validar que el archivo exista.
  • Utilizar rutas organizadas.
  • No sobrescribir información importante accidentalmente.
