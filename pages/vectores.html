Vectores y Matrices en Java

Aprende a almacenar múltiples datos utilizando arreglos unidimensionales y bidimensionales.

Introducción

Los vectores y matrices permiten almacenar varios valores dentro de una sola variable.

Son estructuras fundamentales para resolver problemas relacionados con listas de datos, estadísticas, videojuegos, inventarios y mucho más.

¿Qué es un Vector?

Un vector es un arreglo que almacena múltiples elementos del mismo tipo. 

int[] numeros = {10,20,30,40,50};

Cada posición del arreglo posee un índice. 

Índice:  0   1   2   3   4
Valor:  10  20  30  40  50

Declaración de un Vector

int[] edades = new int[5];

Se crea un arreglo para almacenar 5 enteros.

Asignar Valores

edades[0] = 18;
edades[1] = 20;
edades[2] = 22;
edades[3] = 19;
edades[4] = 25;

¿Qué es una Matriz?

Una matriz es un arreglo bidimensional.

Tiene filas y columnas. 

int[][] matriz = {

    {1,2,3},

    {4,5,6},

    {7,8,9}

};

Representación

1 2 3

4 5 6

7 8 9

Recorrer una Matriz

for(int fila=0;
    fila<3;
    fila++){

    for(int columna=0;
        columna<3;
        columna++){

        System.out.print(
            matriz[fila][columna]
            + " "
        );

    }

    System.out.println();

}

Resumen

  • Un vector almacena múltiples datos del mismo tipo.
  • Una matriz posee filas y columnas.
  • Los arreglos utilizan índices que comienzan en 0.
  • Los ciclos son fundamentales para recorrer arreglos.
  • Los arreglos permiten realizar búsquedas y cálculos rápidamente.
