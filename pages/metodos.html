Métodos (Funciones) en Java

Aprende a crear, reutilizar y organizar código mediante métodos.

Introducción

Un método es un bloque de código que realiza una tarea específica. Gracias a los métodos podemos reutilizar instrucciones sin escribirlas varias veces.

Los métodos permiten que nuestros programas sean más organizados, fáciles de entender y mantener.

¿Qué es un método?

Un método es una agrupación de instrucciones que puede ejecutarse cuando sea necesario. 

public static void nombreMetodo(){

    instrucciones;

}

Ventajas de los Métodos

  • Evitan repetir código.
  • Facilitan el mantenimiento.
  • Permiten dividir problemas grandes.
  • Mejoran la legibilidad.
  • Favorecen la reutilización.

Método Sin Parámetros

Son métodos que ejecutan una tarea sin recibir datos. 

public class Ejemplo {

    public static void saludo(){

        System.out.println("Hola Mundo");

    }

    public static void main(String[] args){

        saludo();

    }

}

Métodos con Parámetros

Los parámetros permiten enviar información al método. 

public class Ejemplo {

    public static void saludar(String nombre){

        System.out.println(
            "Hola " + nombre
        );

    }

    public static void main(String[] args){

        saludar("Christopher");

    }

}

Métodos con Retorno

Un método puede devolver un resultado usando la palabra reservada return. 

public class Ejemplo {

    public static int sumar(
        int a,
        int b
    ){

        return a + b;

    }

    public static void main(String[] args){

        int resultado =
        sumar(5,3);

        System.out.println(resultado);

    }

}

Explicación Paso a Paso

  1. Se crea el método sumar.
  2. Recibe dos números.
  3. Realiza la suma.
  4. Devuelve el resultado.
  5. El método main utiliza el resultado.

Sobrecarga de Métodos

Java permite crear varios métodos con el mismo nombre siempre que tengan parámetros diferentes. 

public class Ejemplo {

    public static int sumar(
        int a,
        int b
    ){

        return a + b;

    }

    public static double sumar(
        double a,
        double b
    ){

        return a + b;

    }

}

Ejemplo Básico Comentado

public class Cuadrado {

    public static int elevarCuadrado(
        int numero
    ){

        return numero * numero;

    }

    public static void main(String[] args){

        int resultado =
        elevarCuadrado(5);

        System.out.println(
            resultado
        );

    }

}

Este método recibe un número y devuelve su cuadrado.

Ejemplo Práctico

Calcular el IVA

public class IVA {

    public static double calcularIVA(
        double precio
    ){

        return precio * 0.15;

    }

    public static void main(String[] args){

        double iva =
        calcularIVA(100);

        System.out.println(
            "IVA: " + iva
        );

    }

}

Este método permite reutilizar el cálculo del IVA para cualquier producto.

Ejercicios Propuestos

  1. Crear un método que muestre tu nombre.
  2. Crear un método que reciba dos números y muestre su suma.
  3. Crear un método que calcule el área de un rectángulo.
  4. Crear un método que determine si un número es par o impar.
  5. Crear un método que reciba una edad y devuelva si es mayor de edad.

Soluciones

Ejercicio 1

public static void mostrarNombre(){

    System.out.println(
        "Christopher"
    );

}

Ejercicio 2

public static void sumar(
    int a,
    int b
){

    System.out.println(a+b);

}

Ejercicio 3

public static double areaRectangulo(
    double base,
    double altura
){

    return base * altura;

}

Ejercicio 4

public static boolean esPar(
    int numero
){

    return numero % 2 == 0;

}

Ejercicio 5

public static boolean mayorEdad(
    int edad
){

    return edad >= 18;

}

Resumen

  • Los métodos permiten reutilizar código.
  • Pueden recibir parámetros.
  • Pueden devolver valores.
  • Ayudan a organizar programas grandes.
  • La sobrecarga permite usar el mismo nombre para tareas similares.
